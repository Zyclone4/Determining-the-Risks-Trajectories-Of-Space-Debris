# Determining-the-Risks-Trajectories-Of-Space-Debris
# Project Specification
As the number of satellites and debris in Earth orbit grows, the risk of collisions increases. Even small fragments can travel at astonishing speeds greater than 7 km/s, causing severe damage to many spacecraft and space missions. By creating a software that can display a dashboard, it can help everyone crucial to space missions improve their situational awareness, support collision-avoidance, and contribute to the sustainability of space operations.

# Goal
The goal of this project is to create a program that can generate a dashboard that visualizes debris trajectories and marks high-risk objects from trained data acquired from CelesTrak.
